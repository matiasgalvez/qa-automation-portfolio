# WebdriverIO + Cucumber Framework Summary

Repository: https://github.com/matiasgalvez/qa-wdio-cucumber

## Overview

This project demonstrates a Web UI automation framework using WebdriverIO, Cucumber, JavaScript and the Page Object Model.

The framework focuses on validating critical e-commerce user journeys such as login, add to cart and checkout.

## Tech Stack

- WebdriverIO
- Cucumber
- JavaScript
- Node.js
- Page Object Model
- Allure Reports
- dotenv
- Gherkin

## Implemented Capabilities

- BDD scenarios using Cucumber and Gherkin
- Page Object Model structure
- Environment-based configuration
- Headless and headed execution
- Screenshots on failure
- Allure test reporting
- Cross-platform npm scripts
- Positive and negative UI scenarios

## Example Covered Flows

- Valid login
- Invalid login
- Add product to cart
- Complete checkout
- Validate checkout confirmation
